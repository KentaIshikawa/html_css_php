引数について

引数を実行時に指定できる様にする事によって、同じ関数でも指定する引数に応じて処理内容を変える事が出来る。
const taxin = (kakaku)=>{
    console.log(kakaku*1.1);
}

関数の実行時に、「()」の中にデータを入れる
なお引数を使用する記述をした関数の実行は、引数を省略できない。
taxin(1000);
ー＞1100

※関数taxinの引数に100を入れると、おかしな数値になる。これはjavascriptの謎の仕様なので、対策する必要があるが、ここでは割愛。

引数は、「,」で区切って幾つでも入れる事ができる
const heikin = (num1,num2) =>{
    console.log((num1 + num2) / 2);
}

関数を実行するときは、引数を使う分だけ記述する必要がある
heikin(4,8);
ー＞6
