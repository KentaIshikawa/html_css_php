無名関数について

あらかじめ定義していない関数を、addEventListenerに入れる事もできる。
この場合、関数を定義する事なく記述できる分、楽にはなるが、
関数に名前をつけられないので後で使い回せない。
無名関数と呼ぶ。
let btn = document.querySelector('#btn');


btn.addEventListener('click',()=>{
    let html = document.createElement('p');
    html.innerHTML = 'クリックされた';

    let target = document.querySelector('#content');
    target.appendChild(html);
});
htmlに要素を追加するときは、まず変数にdocument.createElement(タグ名)で要素を作成し、
その後、変数.innerHTMLで、内容を記述する必要がある。
その変数を、出力したい場所のHTMLを取得し、appendChild()で出力する。
