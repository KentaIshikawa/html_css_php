関数の定義と実行方法

関数の定義方法は複数ある。
1.他のプログラムでもよく使われる方法
function 関数名([引数]){
    処理
}

2.javascriptならではの変数に格納する方法
const 関数名 = function([引数]){
    処理
}

3.最近の書き方（アロー関数）
const 関数名 = ([引数])=>{
    処理
}

※どの書き方も引数は省略可能。引数に関する説明は後述

実行方法
関数名([引数]);


function kansu1(){
    console.log('関数1です');
}

const kansu2 = function(){
    console.log('関数2です');
}

const kansu3 = ()=>{
    console.log('関数3です');
}

kansu1();
ー＞関数1です

kansu2();
ー＞関数2です

kansu3();
ー＞関数3です
