イベントの定義方法

イベントが発生した時に実行したい内容を関数で定義する。
const btn_click = () => {
    console.log('ボタンがクリックされました');
}

イベントを発生させる元のHTMlを取得し、変数に格納する。
その際、HTMLの読み込み順に注意する必要がある。
もし、イベントを発生させる元のHTMLが読み込まれる前にjavascriptを実行してしまうと、動かなくなってしまうので、
HTMLが読み込まれた後に処理が実行される様にscriptを制御する必要がある。

let btn = document.querySelector('#btn');

HTMLが格納された変数に、イベントをセットする
変数.addEventListener(イベント名,行いたい処理);

btn.addEventListener('click', btn_click);
