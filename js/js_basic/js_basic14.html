うるう年かどうかを調べるプログラム

もし年数が、4で割り切れたらうるう年、
さらに、もし100で割り切れる場合はうるう年じゃない、
さらに、もし400で割り切れた場合はうるう年。
全ての条件を満たさない時はうるう年じゃない。

const year = 2020;

let message = '';
if(year % 400 == 0){
    mes = year+'年はうるう年';
}else if(year % 100 == 0){
    mes = year+'年はうるう年じゃない';
}else if(year % 4 == 0){
    mes = year+'年はうるう年';
}else{
    mes = year+'年はうるう年じゃない';
}

const text = document.querySelector('#text');
text.innerHTML = mes;
