for文について

書き方
for(カウンターのセット;繰り返す条件;カウンターの処理){
    繰り返す処理
}

let count = 0;

for(count;count<3;count++){
    console.log('繰り返し');
}

ー＞（3）繰り返し
本来なら「繰り返し」という文字が3回出力されるが、同じ言葉なので開発者ツールでは（3）と表示される

なお、一般的にはわざわざカウンターを変数て定義せず、for文と一緒に書くのが普通。
さらに、カウンター用の変数は、「i」を使う人が多い。
カウンター用の変数は、繰り返し処理の中で出力などに使う事ができる。
for(let i;i$lt;3;i++){
    console.log(i);
}
