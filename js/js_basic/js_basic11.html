様々な計算

縦が4、横が6の長方形の面積
const tate = 4;
const yoko = 6;
console.log(tate * yoko);
ー＞24

底辺が6、高さが8の三角形の面積
const tei = 6;
const takasa = 8;
console.log(tei * takasa);
ー＞24

上底が3、下底が5、高さが4の台形の面積

const takasa2 = 4;
console.log((joutei + katei) * takasa2 / 2);
