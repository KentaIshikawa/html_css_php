for文以外の繰り返しの書き方

javascriptのバージョンアップに伴い、様々な書き方で繰り返し処理がかけるようになった。
配列と、各々の繰り返し時の時に使う配列内の値を格納した変数名をセットして、for of 文が使える

for(その繰り返し順の時の配列の値 of 配列){
    処理
}

const drinks =[
    'ビール',
    'ウィスキー',
    '日本酒',
    '麦焼酎'
];

for(drink of drinks){
    console.log(drink);
}
ー＞ビール
ー＞ウィスキー
ー＞日本酒
ー＞麦焼酎
