数値（Number）

いわゆる数字のこと。
半角入力で、'「シングルクォート」や"「ダブルクォート」を使わずに記述する。

const kazu = 1;
console.log(kazu);
ー＞1

四則演算が可能。
足し算
console.log(3+4);
ー＞7

引き算
console.log(5-2);
ー＞3

掛け算
console.log(5*3);
ー＞15

割り算
console.log(6/2);
ー＞3

計算の順番も四則演算通り
console.log(3+2*2);
ー＞7

()がある場合は、そちらを先に計算する
console.log((3+2)*2);
ー＞10

割った時の余りを求める剰余算
console.log(5%2);
ー＞1
