戻り値について

関数の処理の中で、return を書く事で、関数の中で処理した内容を、関数を実行した場所に戻す事ができる。
returnは、「値を戻す」だけでなく「プログラムを止める」という意味もある為、通常関数の処理の中の最後に書かれる事が多い。

const large_num = (num1,num2) => {
    let larger;
    if(num1 > num2){
        larger = num1;
    }else{
        larger = num2;
    }
    return larger;
}

const apple = 120;
const banana = 100;

戻り値がある関数を実行すると、その値を左辺に代入する事ができる。
const num = large_num(apple,banana);
console.log('りんごとバナナの金額の高い方は、' + num + '円です。');

戻り値を設定しておくと、関数を使いまわせる

const tone = 322;
const tama = 138;

const num2 = large_num(tone,tama);
console.log('利根川と多摩川の長い方は、約' + num2 + 'kmです。');
