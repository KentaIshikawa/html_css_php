真偽値とif文

真偽値は、if文の条件に使ったとき、trueだったら条件を通り、falseだったら通らない仕様になっている。

const check = true;

if(check){
    console.log('変数checkには、trueが代入されているので必ず条件は成り立つ');
}

チェック用の変数を作り、if文を制御する際、trueとfalseを交互に入れ替えれば、
一つのボタンを押すたびに処理が変える事ができる。（トグル処理と呼ぶ）

let check2 = true;
let btn = document.querySelector('#btn');

条件分岐のifとelseの処理に、check2の値にtrueとfalseを交互に入れる様に設定する
btn.addEventListener('click', () => {
    if (check2) {
        check2 = false;
        let target = document.querySelector('#text');
        target.classList = 'show';
    } else {
        check2 = true;
        let target = document.querySelector('#text');
        target.classList = '';
    }
});
クラスをつけるには、classListにクラス名を入れる。
逆にクラスを外したいときは、クラス名に空の文字列を入れる。

ここの文字は、ボタンをクリックする事で、出たり消えたりします。
