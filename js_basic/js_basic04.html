変数の使い方の一例

変数は、文章の中の「共通する部分」をデータとして代入しておくと、合理的に作業ができる。

例）
鈴木さん
佐藤さん
田中さん
以上のような文章があるとして、敬称である「さん」を「君」に直す場合、
三箇所の「さん」を「君」に直さなくてはならない。

鈴木君->1回
佐藤君->2回
田中君->3回

これがもし、300人分直そうと思ったら、300箇所直さなくてはならない。

もし敬称である「さん」を変数をつかって書いていた場合、、、

const kei = 'さん';
'斎藤' + kei;
'伊藤' + kei;
'山本' + kei;

以上のような文章で、「さん」を「君」に直そうとすると、

const kei = '君';
'斎藤' + kei;
'伊藤' + kei;
'山本' + kei;

一箇所だけ直せば良い。
仮に、300人分直そうと思っても、一箇所だけ直せば良い。
