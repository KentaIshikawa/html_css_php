変数の種類について

変数には、var、let、constの三種類の定義方法がある

変数の作り方
var 変数名
or
let 変数名
or
const 変数名;

各々の違い
varは、従来からある方法、変数の有効範囲が広いため、名前が被らないように注意しなくてはならない。
letは、割と最近に追加された方法。有効範囲が狭く、名前が被りにくく使い勝手が良い。
constは、let同様有効範囲が狭く、変更も不可。もっとも使いやすい。
基本的に変数は、使い方が限られている方が使いやすい。
逆に、使い方が自由であればあるほど、間違えないように気を配る箇所が増える。

基本的には、変数の有効範囲（スコープという）が狭い順に使う、
変数の値を変更する必要がない時は「const」、
変更する必要がある時や、追記する必要がある時は「let」、
スコープをプログラム全域（グローバル変数）とする必要がある時は「var」、
この順番でOK。

現時点では、少し難しすぎる考え方なので、以下のように考えてOK
const > let > var
の順でOK

変数の追記について

let mozi = '<p>';
    mozi += 'こんにちは';
    mozi += '</p>';
    console.log(mozi);
    ＝＞<p>こんにちは</p>が出力される
