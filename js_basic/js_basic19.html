配列と繰り返し

配列と繰り返し処理は相性がよい。
配列に出力したいデータを格納しておいて、一気に出力する事が出来る。

const foods =[
    'パスタ',
    'ハンバーグ',
    'カレー',
    'ステーキ'
];

for(let i = 0;i<4;i++){
    console.log(foods[i]);
};
ー＞パスタ
ー＞ハンバーグ
ー＞カレー
ー＞ステーキ

なお、配列の中に何個データが入っているかは、
配列名.length
で取得できるため、配列の中のデータを全て出力する場合は、lengthを使うのが一般的。

for(let i = 0;i<foods.length;i++){
    console.log(foods[i]);
}
ー＞パスタ
ー＞ハンバーグ
ー＞カレー
ー＞ステーキ
