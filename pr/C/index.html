販売員としての実績

私は、大学卒業から6年間、ジュエリーや雑貨を販売するお店で働いてきました。
お客様をよく観察し、お声がけする事で売り上げをあげられる様努力しました。
また、一部の商品の発注も担当し、世の中のトレンドに合わせて、人気のものを仕入れる様にしていました。

好きなことには夢中になる性格です

3年ほど前からジムに通い出し、週2で通っています。最近では、ジムだけではなく地域のハーフマラソンへ参加したりしています。今後の目標は、東京マラソンでフルマラソン走りきりことです。
